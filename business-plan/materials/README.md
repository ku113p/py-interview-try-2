# Shared Assets

This directory stores shared assets used across all stages: logos, screenshots, diagrams, videos.

**Actionable materials (templates, scripts, checklists) live inside their stage directories.**

---

## Asset Types

### Visual Assets
- Logo (when created)
- Product screenshots (blur private data)
- Architecture diagrams
- Icons / social media images

### Video Assets
- Demo recordings (raw + edited)
- Tutorial videos
- Upload to YouTube, link from docs

### Documents
- Competitor research (when done)
- Customer interview notes
- Usage analytics exports
